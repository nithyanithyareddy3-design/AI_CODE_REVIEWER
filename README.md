# 💻 AI Code Reviewer

An AI-powered code review application built with **Python**, **Streamlit**, and the **Groq API**. It analyzes source code, detects bugs, explains logic, identifies code smells, and suggests improvements to help developers write cleaner, more efficient, and maintainable code.

## 🚀 Features

* Review code using AI
* Supports multiple programming languages
* Detects bugs and potential errors
* Suggests performance and readability improvements
* Recommends coding best practices
* Provides clear explanations of the code
* Simple and interactive Streamlit interface

## 🛠️ Technologies Used

* Python
* Streamlit
* Groq API
* python-dotenv

## ⚙️ Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd AI_CODE_REVIEWER
```

2. Create and activate a virtual environment:

```bash
python -m venv venv
```

Windows:

```bash
venv\Scripts\activate
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Create a `.env` file and add your API key:

```env
GROQ_API_KEY=your_groq_api_key
```

5. Run the application:

```bash
streamlit run app.py
```
